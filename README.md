# Worklet Playground
A sandbox app to test building and running Worklets in Typescript, in an app bootstrapped with Create React App.

Uses the [new syntax](https://github.com/webpack/webpack/pull/17212) introduced by @alexander-akait in [Webpack 5.85](https://github.com/webpack/webpack/releases/tag/v5.85.0).

Discussion in the thread around [this entry](https://github.com/webpack/webpack/issues/11543#issuecomment-1585715108).
